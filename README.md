# Overview

This repo is a wrapper of a <https://github.com/Yog9/SnapShot> photo viewer, sourced from Flickr, wrapped in a Custom Element and provided with the descriptor files necessary to deploy the React application as a widget (snapshot-widget) in Entando.

This application was generated using JHipster 6.9.0, you can find documentation and help at [https://www.jhipster.tech/documentation-archive/v6.9.0](https://www.jhipster.tech/documentation-archive/v6.9.0).
